==================
OpenEEW for Python
==================

This is the documentation for openeew, a Python package for working with all aspects of the OpenEEW initiative by `Grillo <https://grillo.io/openeew/>`_.

Currently the package can be used to simplify selecting and downloading OpenEEW data stored as an `AWS Public Dataset <https://registry.opendata.aws/grillo-openeew/>`_.
